# Atlas AI Labs — Experience Website

Apresentação interativa para convidar o Dr. Allan Rabello a ser Clínica Fundadora da DentalFlow AI.

## Como publicar rapidamente

### Opção 1: Vercel
1. Crie uma conta em https://vercel.com
2. Clique em "Add New Project"
3. Envie esta pasta ou conecte a um repositório GitHub
4. Publique

### Opção 2: Netlify
1. Crie uma conta em https://netlify.com
2. Clique em "Add new site" > "Deploy manually"
3. Arraste a pasta `atlas_experience`
4. O link será gerado automaticamente

### Opção 3: GitHub Pages
1. Crie um repositório
2. Envie os arquivos
3. Ative GitHub Pages nas configurações do repositório

## Arquivos
- `index.html`: conteúdo da apresentação
- `styles.css`: visual e responsividade
- `script.js`: animações de entrada e efeito de cursor
