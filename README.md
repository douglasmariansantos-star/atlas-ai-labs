# Atlas AI Labs — Experience Website v2

Esta é a versão cinematográfica da apresentação da Atlas AI Labs para convidar o Dr. Allan Rabello a ser a primeira Clínica Fundadora da DentalFlow AI.

## Arquivos

- `index.html` — estrutura da página
- `styles.css` — visual, responsividade e animações
- `script.js` — animações ao rolar e brilho do cursor

## Como publicar no GitHub Pages

1. Envie estes arquivos para o repositório `atlas-ai-labs`.
2. Vá em **Settings → Pages**.
3. Em **Source**, escolha **Deploy from a branch**.
4. Em **Branch**, escolha `main` e `/root`.
5. Clique em **Save**.

## Como atualizar o site

Substitua os arquivos antigos por estes, faça commit no GitHub Desktop e clique em **Push origin**.
